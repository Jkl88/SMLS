# Создание репозитория на GitHub для SMLS

Локальный Git-репозиторий уже инициализирован, коммиты есть.

## Как создать репозиторий на GitHub и выгрузить код

### 1. Создать репозиторий на GitHub

1. Откройте [github.com](https://github.com) и войдите в аккаунт.
2. Нажмите **«+»** → **«New repository»**.
3. Укажите имя, например: **SMLS** (или другое).
4. Оставьте репозиторий **пустым**: не добавляйте README, .gitignore и лицензию.
5. Нажмите **«Create repository»**.

### 2. Подключить удалённый репозиторий и отправить код

В каталоге проекта выполните (подставьте свой логин и имя репозитория):

```bash
cd c:\Users\Oleg\PlatformIO\SMLS

git remote add origin https://github.com/ВАШ_ЛОГИН/SMLS.git
git push -u origin master
```

Если репозиторий уже был добавлен и нужно только отправить код:

```bash
git push -u origin master
```

### 3. Проверить

После `git push` код появится на странице репозитория на GitHub.
